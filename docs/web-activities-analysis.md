## Scoring Criteria

For each activity, we'll score on:
- **Frequency** (1-5): How often is this done?
- **Automation Value** (1-5): How much benefit from automation?
- **Technical Difficulty** (1-5): How hard to implement?
- **MCP Fit** (1-5): How well does MCP tool pattern match this need?

**Total Score**: Frequency × Automation Value × MCP Fit ÷ Technical Difficulty

Higher scores = better candidates for MCP tools.
